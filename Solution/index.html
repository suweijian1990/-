星羽創媒是一家以 AI 技術為核心驅動的企業級數字營銷與智能內容基礎設施公司，致力於建構面向下一代內容產業的智能生產系統，推動企業從傳統數字化內容生產模式邁向 AI 原生驅動的系統化生產階段。

公司由山桃文化創始人陳嘉勝創立。在早期創業實踐中，山桃文化圍繞微短劇、品牌營銷及文化出海等領域，持續探索數字化內容生產與技術工具化能力，在內容生產流程管理、項目協同與營銷系統化方面形成了一定的技術與方法積累，為後續的 AI 系統化升級提供了實踐基礎。

在產生式 AI 技術加速發展的背景下，基於對內容產業智能化演進趨勢的判斷，陳嘉勝進一步推動業務升級，創立星羽創媒，將原有數字化內容生產能力全面升級為 AI 原生內容與營銷系統能力。

星羽創媒圍繞AI Studio 一站式創意平台、企業級 AI Agent 智能體系統，以及無限畫布工作流引擎三大核心產品體系，建構覆蓋創意策劃、視覺設計、影片生產、內容分發與企業運營的智能生產網絡。

通過將 AIGC 能力深度嵌入企業內容與營銷全鏈路，星羽創媒實現從"數字化流程驅動"向"AI系統驅動"的結構性躍遷，致力於成為企業級 AI 內容與營銷基礎設施提供者。

Solution

項目技術方案

採用「統一平台 + 四大模組 + 獨立基礎能力層」的架構設計
各模組通過統一權限中心與 API 閘道進行存取控制，前端統一門戶按角色展示

MODULE 01

輿情儀表板模組

對接 Facebook、Instagram、OpenRice、小紅書等社交平台 API，定時拉取十幾個品牌、300+ 門店的使用者評論與帖文，匯入統一情感分析引擎。

  • 負面評論自動標紅，結合品牌 SOP 範本產生回覆草稿，人工確認後傳送
  • 正面評論自動產生感謝回覆，中性評論提供標準化話術
  • 處理建議通過 API 推送至客戶自有系統，不新建工單模組
  • 按門店-品牌架構表自動路由分配責任人
  • 前期選一個品牌下門店做試點驗證
各平台 API 適配 粵語/中英混合情感分析 封閉平台數據採集
MODULE 02

智慧問數模組

基於 NL2SQL（自然語言轉 SQL）技術路線。系統不直接存取客戶資料庫，通過語義映射層建構 Schema Linking，保障數據安全。

  • 自然語言提問 → 語義解析（指標/維度/時間/聚合） → SQL 產生 → 圖表渲染
  • AI 對查詢結果給出數據洞察與建議
  • 第一優先級：固定駕駛艙（預設 KPI 卡片和圖表）
  • 第二階段：開放自定義儀表板設定，可與 PowerBI 平行或嵌入整合
  • 按品牌/區域權限自動注入數據範圍過濾條件，確保數據隔離
NL2SQL 準確率最佳化 Schema Linking 建構
MODULE 03

工程設計圖模組

以歷史 CAD 設計圖作為訓練素材，歸納各品牌設計規範範本（燈光、動線、材質等），建構品牌設計知識函式庫。

  • 輸入新建築師 DWG 圖紙 → AI 產生施工圖級別 CAD 檔案
  • 人機協作：線上預覽 + 自然語言描述修改需求
  • 支援版本管理和修改追溯
  • 香港消防條例、建築規範等法規數字化合規校驗
  • 合規檢查作為輔助參考，最終以專業工程師審核為準
施工圖精度要求高 AI 輔助設計定位
